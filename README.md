# Ammajaan - E-Commerce Product Explorer

React + Vite e-commerce project built from PRD requirements with a neo-brutalist UI direction.

## Stack

- React
- Vite
- React Router DOM
- Axios
- React Hook Form + Yup
- React Toastify
- Framer Motion
- Swiper

## Run Locally

1. Install dependencies

   npm install

2. Start development server

   npm run dev

3. Build for production

   npm run build

4. Run lint checks

   npm run lint
